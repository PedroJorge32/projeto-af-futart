# FutArt

## Tema
Site fictício de uma loja de artigos esportivos focada em camisas de futebol, lançamentos e produtos para torcedores.

## Integrantes
- Pedro Henrique dos Santos Jorge - RA: 250972

## Páginas do Projeto
- Home
- Galeria
- Contato

## Tecnologias Utilizadas
- HTML5
- CSS3
- Bootstrap
- JavaScript
- GitHub

## Funcionalidades
- Menu responsivo com Bootstrap Navbar
- Carrossel Bootstrap na página inicial
- Galeria de imagens com cards e filtros por categoria
- Formulário de contato com validações HTML5
- Mapa incorporado na página de contato
- Botão voltar ao topo
- Efeitos visuais em cards, botões e imagens
- Layout responsivo com Bootstrap Grid e Flexbox
